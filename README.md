Гатауллин Алмаз Мансурович

Приложение для поиска пользователей GitHub с возможностью добавления их в избранное и сохранения списка в JSON.
1. Введите имя пользователя в поле поиска.
2. Нажмите «Поиск».
3. Для добавления в избранное дважды кликните по пользователю в списке.
4. Избранные пользователи сохраняются в `favorites.json`.
- Python 3.x
- Tkinter (GUI)
- GitHub API (REST)
- JSON (хранение данных)
- Git (контроль версий)

Пример использования
1. Введите `torvalds` и нажмите «Поиск».
2. Дважды кликните по результату — Линус Торвальдс будет добавлен в избранное.
3. Проверьте файл `favorites.json`.
